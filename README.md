# 🧠 XAI Stress Detection Dashboard
An Explainable AI (XAI) dashboard for student stress detection using AdaBoost classifier with SHAP explanations. Built with Streamlit.

## Overview
This dashboard provides an interactive interface for stress level prediction using machine learning with explainable AI capabilities. It helps students and educators understand stress patterns and provides personalized recommendations based on individual responses.

## Features
- **AdaBoost Classifier** - Machine learning model for stress prediction
- **SHAP Explanations** - Understand why the model makes specific predictions
- **Interactive Questionnaire** - User-friendly stress assessment with 0-10 scale
- **Personalized Recommendations** - Tailored advice based on stress levels
- **PDF Report** - Downloadable PDF report as the record 

## Run the Application
The application will open in your default browser using the below code
streamlit run dashboard.py
